1. Benefits

Use file system that is collection of files and folders as note managing style program.

  • it remains foever and everywhere accessible
  • no problem of unsupported softwares
  • can mix different file types like org,txt,html,doc,…

2. How?

  • Make a convention for file naming:

YYMMDD-X-NAME

  • First part is timestamp. Its good to order file by a timestamp as humans are good at remebering dates of events (Noguchi Filing System)
  • X is a broad category identifier. e.g:
    • R for record
    • J for journal
    • N for notes
    • P for project related notes …
  • X may further be subcategorized as X1,X2,…
  • NAME is an unique identifying name
  • Further the files can be arranged in cateogrical folders and sub-folders as per need.
