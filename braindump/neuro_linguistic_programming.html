3.1. Color Coding

Assign each subject an unique color and use that color for that task. So, e.g. using green color for science topic, use green copy, green pen, … then in science exam using green triggers (like eraser, cloth, ..) brings mind to science phase and recalling & performance is boosted

3.2. Music
